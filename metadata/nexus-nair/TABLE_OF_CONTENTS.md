### 5.2. `brands` Table Schema

**Full SQL Schema:**
```sql
CREATE TABLE brands (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    name VARCHAR(255) UNIQUE NOT NULL,
    tier INTEGER CHECK(tier BETWEEN 1 AND 5) NOT NULL,
    organization VARCHAR(50) CHECK(organization IN ('FAA', 'HSOMNI', 'Seedwave')) NOT NULL,
    category VARCHAR(100) NOT NULL,
    status VARCHAR(20) DEFAULT 'active' CHECK(status IN ('active', 'inactive', 'pending', 'suspended')),
    ecr_fee_annual DECIMAL(10,2),
    royalty_percentage DECIMAL(5,2),
    region VARCHAR(50),
    country VARCHAR(100),
    continent VARCHAR(50),
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

CREATE INDEX idx_brands_tier ON brands(tier);
CREATE INDEX idx_brands_organization ON brands(organization);
CREATE INDEX idx_brands_category ON brands(category);
CREATE INDEX idx_brands_country ON brands(country);
```
